This terraform script will create a Ubuntu server and install apache2 and the necessary infrastructure e.g VPC, route table, security group etc.

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

1. [Terrafom CLI](https://www.terraform.io/downloads)
2. [AWS Account](https://aws.amazon.com/)
  
### Installation

1. Clone the repo *Project 1*
2. Update the following
   ```sh
   AWS cli credentials in provider.tf 
   ```
3. Run Terraform apply
